# Docker Project

## Docker Image
- **Docker Hub Repository**: [Docker Hub Link](https://hub.docker.com/repository/docker/oma910/Docker)
- **Usage**:
  1. Pull the Docker image:
     ```bash
     docker pull oma910/Docker:latest
     ```
  2. Run a container:
     ```bash
     docker run -p 3000:3000 oma910/Docker:latest
     ```
